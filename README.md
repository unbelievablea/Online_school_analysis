# Анализ данных онлайн-школы

Учебный проект по анализу эффективности маркетинговых кампаний и поведения студентов.

## Задача

Проанализировать данные онлайн-школы, чтобы ответить на вопросы:
- Какие каналы привлечения наиболее эффективны?
- Какова конверсия из бесплатного взаимодействия в платное?
- На каких этапах воронки теряется больше всего пользователей?

## Ход работы

1. **Выгрузка сырых данных** — получение логов о взаимодействии пользователей
2. **Определение последнего платного клика** (`last_paid_click.sql`) — для каждого пользователя находится момент, когда он совершил платное действие
3. **Агрегация** (`aggregate_last_paid_click.sql`) — подсчёт количества платных кликов по источникам трафика
4. **Формирование дашборда** (`dashboard.sql`) — итоговая витрина для визуализации

## Результаты

Основные выводы представлены в файле [`presentation.pdf`](./presentation.pdf).

Ключевые метрики, которые удалось рассчитать:
- Конверсия в платное действие по каждому каналу
- Распределение платящих пользователей по источникам
- Точки отсечения в воронке

## Как воспроизвести

1. Установите PostgreSQL
2. Импортируйте сырые данные (CSV) в базу
3. Последовательно выполните SQL-файлы в порядке нумерации
4. Результаты сохранятся в CSV-файлах
