# Анализ данных онлайн-школы

Учебный проект по анализу эффективности маркетинговых кампаний и поведения студентов.

## Задача

Проанализировать данные онлайн-школы, чтобы ответить на вопросы:
- Какие каналы привлечения наиболее эффективны?
- Какова конверсия из бесплатного взаимодействия в платное?
- На каких этапах воронки теряется больше всего пользователей?
